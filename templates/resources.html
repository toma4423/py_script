PyScriptとは

PyScriptは、HTMLドキュメント内でPythonコードを直接実行できるようにするフレームワークです。WebAssemblyを利用して、ブラウザ上でPythonコードを実行します。これにより、JavaScriptを使わずにPythonだけでインタラクティブなWebアプリケーションを作成することが可能になります。

主な特徴

  • HTMLファイル内に直接Pythonコードを記述できる
  • NumPy、Pandas、Matplotlib、scikit-learnなどの人気のあるPythonライブラリを使用できる
  • DOMの操作が可能
  • 外部APIとの通信が可能
  • データの可視化が簡単に行える

サンプルコード

基本的な使い方

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="https://pyscript.net/latest/pyscript.css" />
    <script defer src="https://pyscript.net/latest/pyscript.js"></script>
</head>
<body>
    <py-script>
        print("Hello, PyScript!")
    </py-script>
</body>
</html>

外部パッケージの使用

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="https://pyscript.net/latest/pyscript.css" />
    <script defer src="https://pyscript.net/latest/pyscript.js"></script>
</head>
<body>
    <py-config>
        packages = ["numpy", "matplotlib"]
    </py-config>
    
    <div id="plot"></div>
    
    <py-script output="plot">
        import numpy as np
        import matplotlib.pyplot as plt
        
        x = np.linspace(0, 10, 100)
        y = np.sin(x)
        
        plt.figure(figsize=(8, 4))
        plt.plot(x, y)
        plt.title('サイン関数')
        plt.grid(True)
        
        plt
    </py-script>
</body>
</html>

DOMの操作

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="https://pyscript.net/latest/pyscript.css" />
    <script defer src="https://pyscript.net/latest/pyscript.js"></script>
</head>
<body>
    <input id="name-input" type="text" placeholder="あなたの名前">
    <button id="greet-button" pys-onClick="greet">挨拶する</button>
    <div id="output"></div>
    
    <py-script>
        from pyscript import Element
        
        def greet(*args, **kwargs):
            name = Element("name-input").value
            if not name:
                name = "ゲスト"
            Element("output").write(f"こんにちは、{name}さん！")
    </py-script>
</body>
</html>

FastAPIとPyScriptの連携

FastAPIとは

FastAPIは、Pythonで高速なAPIを構築するための最新のWebフレームワークです。自動ドキュメント生成、型チェック、非同期処理などの機能を備えています。

PyScriptとFastAPIの連携例

<!-- HTMLファイル内のPyScript部分 -->
<py-script>
    from pyodide.http import pyfetch
    from pyscript import Element
    
    async def call_api(*args, **kwargs):
        try:
            # FastAPIエンドポイントを呼び出す
            response = await pyfetch(
                url="/api/data",
                method="GET"
            )
            
            if response.status == 200:
                data = await response.json()
                Element("output").write(f"APIからのデータ: {data}")
            else:
                Element("output").write(f"エラー: {response.status}")
        
        except Exception as e:
            Element("output").write(f"エラー: {str(e)}")
</py-script>

よくある質問

PyScriptはどのブラウザで動作しますか？

PyScriptは最新のモダンブラウザ（Chrome、Firefox、Edge、Safariなど）で動作します。ただし、WebAssemblyをサポートしているブラウザが必要です。

PyScriptのパフォーマンスはどうですか？

PyScriptはWebAssemblyを使用してPythonコードを実行するため、ネイティブのPythonよりは遅くなります。特に初回ロード時には時間がかかることがあります。ただし、一度ロードされると、比較的スムーズに動作します。

すべてのPythonライブラリが使用できますか？

すべてのPythonライブラリが使用できるわけではありません。特にC拡張を使用するライブラリは動作しない場合があります。ただし、NumPy、Pandas、Matplotlib、scikit-learnなどの多くの人気ライブラリはサポートされています。

PyScriptでファイルの読み書きはできますか？

ブラウザのセキュリティ制限により、ローカルファイルシステムへの直接アクセスはできません。ただし、ブラウザのFileSystemAPIを使用したり、サーバーとの通信を通じてファイル操作を行うことは可能です。

JavaScriptを全く使わずにWebアプリケーションを作れますか？

はい、PyScriptとFastAPIを組み合わせることで、JavaScriptを直接書かずにPythonだけでインタラクティブなWebアプリケーションを構築することができます。ただし、内部的にはPyScriptがPythonコードをJavaScriptに変換しています。
